# _Triangles_

##### _Returns type of triangle given side lengths, August 10th, 2015_

#### By _**Clayton Easterbrooks & Vaughn Dice**_

## Description

_Returns type of triangle given side lengths if it is a triangle_

## Setup

* _clone this repository_
* _run `bundle`_
* _run `ruby app.rb` to serve web app locally_
* _navigate to `localhost:4567`
* _run `rspec` to run all tests

## Technologies Used

_Ruby, rspec, Sinatra, Capybara, pry_
